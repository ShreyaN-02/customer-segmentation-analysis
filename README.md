# customer-segmentation-analysis
Customer segmentation analysis using SQL, Excel and Power BI with clustering-based insights 

Overview
This project focuses on segmenting customers based on demographic and behavioral attributes using clustering techniques. The goal is to identify distinct customer groups and generate actionable business insights.

Tools Used:
Excel (Data Cleaning)
SQL (Data Analysis)
Power BI (Visualization & Dashboard)

Dataset:
The dataset includes:
Customer ID
Gender
Age
Annual Income
Spending Score

Key Features:
Customer segmentation using income and spending behavior
Revenue Potential Score calculation
Interactive Power BI dashboard
Business insights and recommendations

Key Insights:
Standard customers form the largest segment
Premium customers generate highest revenue potential
Conservative customers represent upselling opportunities

Business Recommendations:
Loyalty programs for premium customers
Targeted promotions for conservative segment
Discount strategies for budget customers
